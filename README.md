# CareerMatch AI 求职智能匹配助手

这是一个可部署公网的比赛版 Demo，面向学生求职场景，支持简历与岗位JD匹配分析、岗位匹配度评分、学生画像生成、关键词匹配、简历优化建议和投递策略输出。

## 本地运行

直接双击 `index.html` 即可运行。

## Vercel 部署

1. 将整个项目文件夹上传到 GitHub 仓库。
2. 打开 https://vercel.com 并登录。
3. 点击 Add New Project，选择该仓库。
4. Framework Preset 选择 Other。
5. 点击 Deploy。
6. 部署完成后复制公网链接提交。

## Netlify 部署

1. 打开 https://app.netlify.com/drop。
2. 将整个项目文件夹拖入页面。
3. 自动生成公网链接。

## 文件说明

- `index.html`：页面结构
- `style.css`：页面样式
- `app.js`：匹配分析逻辑
- `scheme.md`：1000字以内方案说明
